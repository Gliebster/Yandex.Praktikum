1. Восстанавливаем базу из резервной копии:
переносим файл дампа в папку /tmp, которая доступна всем пользователям:
cp /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/my_zen.dump /tmp

2. Восстанавливаем данные базы из дампа (при условии, что БД уже существует):
pg_restore -d zen /tmp/my_zen.dump

3. Запустить пайплайн можно следующей командой:
python3 /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/zen_pipeline.py --start_dt='2019-09-24 18:00:00' --end_dt='2019-09-24 19:00:00'

4. Запустить дашборд можно следующей командой:
python3 /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/dashboard.py
далее набрать в браузере: localhost:8050
в фильтре по времени дату окончания стоит установить 2019-09-25
