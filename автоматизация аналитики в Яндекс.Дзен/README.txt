**Задание**
Вы работаете аналитиком в Яндекс.Дзене. Почти всё ваше время занимает анализ пользовательского взаимодействия с карточками статей.
Каждую карточку определяют её тема и источник (у него тоже есть тема). Примеры тем: «Красота и здоровье», «Россия», «Путешествия».
Пользователей системы характеризует возрастная категория. Скажем, «26-30» или «45+».

Есть три способа взаимодействия пользователей с системой:
Карточка отображена для пользователя (show);
Пользователь кликнул на карточку (click);
Пользователь просмотрел статью карточки (view).

Каждую неделю начинающие менеджеры Денис и Валерия задают вам одни и те же вопросы:
Сколько взаимодействий пользователей с карточками происходит в системе с разбивкой по темам карточек?
Как много карточек генерируют источники с разными темами?
Насколько хорошо пользователи конвертируются из показов карточек в просмотры статей?
На шестую неделю работы вы решаете, что процесс пора автоматизировать. Для Дениса и Валерии нужно сделать дашборд.

** Примечание **
В файле zen_pipeline.py содержится дата-пайплайн, который берет первичные данные из базы, строит таблицы с агрегированными данными, которые будут использоваться для построения дашборда, и записывает их в БД. В файле dashboard.py содержится код дашборда. На дашборде отображен график истории событий по темам карточек, график разбивки событий по темам источников, график средней глубины взаимодействия (без разбивки на темы карточек и возрастные категории).


** Напоминалочка о том, как все правильно запустить **
1. Восстанавливаем базу из резервной копии:
переносим файл дампа в папку /tmp, которая доступна всем пользователям:
cp /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/my_zen.dump /tmp

2. Восстанавливаем данные базы из дампа (при условии, что БД уже существует):
pg_restore -d zen /tmp/my_zen.dump

3. Запустить пайплайн можно следующей командой:
python3 /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/zen_pipeline.py --start_dt='2019-09-24 18:00:00' --end_dt='2019-09-24 19:00:00'

4. Запустить дашборд можно следующей командой:
python3 /ПУТЬ_К_ПАПКЕ_С_ФАЙЛОМ/dashboard.py
далее набрать в браузере: localhost:8050
в фильтре по времени дату окончания стоит установить 2019-09-25
